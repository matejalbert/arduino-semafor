# Arduino Semafor

Jednoduchý projekt semaforu postavený na Arduino desce.  
Semafor střídá světla zelenou, žlutou a červenou pomocí LED diod, aby simuloval běžný dopravní semafor.

---

## Funkce

- střídání tří LED diod (zelená, žlutá, červená)  
- jednoduchý časovač pro každou barvu  
- snadné přizpůsobení délky svícení jednotlivých světel
